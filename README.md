\# Java TCP Guessing Game



Aplicación desarrollada en Java que implementa un juego cliente-servidor mediante sockets TCP.



\## 📖 Descripción



Este proyecto consiste en un juego de adivinar un número en el que un servidor gestiona la partida y uno o varios clientes intentan descubrir el número secreto.



La comunicación entre el cliente y el servidor se realiza mediante sockets TCP y los mensajes intercambiados se cifran utilizando AES.



\## 🚀 Características



\- Arquitectura cliente-servidor.

\- Comunicación mediante sockets TCP.

\- Cifrado de mensajes con AES.

\- Organización del código en paquetes.

\- Desarrollo realizado en Java.



\## 🛠️ Tecnologías utilizadas



\- Java

\- TCP Sockets

\- AES Encryption

\- Programación concurrente
